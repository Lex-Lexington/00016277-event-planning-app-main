# 00016277-event-planning-app

Event planning app

About:

This program is designed to simplify the planning and organization of various events.

Instruction:

1. After opening the application, open a terminal
2. Run: npm init -y
3. Then install dependencies: npm i express express-validator pug body-parser nodemon
4. Start the app: npm start
5. Open http://localhost:3000

Application dependencies:

1. body-parser
2. express
3. express-validator
4. nodemon
5. pug
